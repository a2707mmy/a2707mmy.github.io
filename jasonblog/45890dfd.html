4.多邊形:

函數原型:

範例:

因為此函式設計成可以同時畫出多個多邊形，因此相對來說複雜許多，傳入的資料型態多為陣列。

說明:

  1. 首先，所有資料點會形成一個二維陣列，而每一列的資料點又會形成一個多邊形，因此可想而知，此函數若要分辨每個多邊形由哪些點所組成，除了所有資料點要傳入之外，還需要知道每一個多邊形有幾個資料點，總共有幾個多邊形。
  2. 第二個引數是所有資料點的指標。也就是上述所說的所有資料點形成的二維陣列指標。
  3. 第三個引數代表每個多邊形由幾個點所構成，這個參數也是個陣列。
    例如: int arr[3] = { 4,3,4 };
    代表第一個多邊形共有四個點，第二個多邊形有三個點，第三個多邊形有四個點。
  4. 第四個參數是代表有幾個多邊形(根據函式說明是幾個連線)。
  5. 第五個參數是代表多邊形折線最後一點是否要連回第一點形成封閉多邊形。
