本篇文章記錄OpenCV中的Mat物件，及讀取圖像、輸出圖像及獲取特定像素的RGB資料的方式。
在注重圖像格式的時刻詳細的了解Mat物件的各項屬性及前述內容，有助於讓使用者精確的操作圖像及像素。
你可以在這篇文章中學到以下內容

  1. Mat物件
  2. imread
  3. imwrite
  4. 獲取特定像素的RGB資料

  3. int rows:像素陣列的列數。
  4. int cols:像素陣列的行數。
  5. uchar* data:uchar類型的指標，指向像素陣列的首個地址。
  6. int dims:陣列的維度，以圖像來說為2。

三、 常用函數:

  1. int channels( ):回傳通道數，1為灰度圖， 3為BGR圖， 4為BGRA圖(A為透明度)。
  2. int depth( ):回傳一整數，用以表示像素的精度。
    表一、depth()回傳的整數與像素資料型態的對應關係

    | 8S=1 | 16S=3 | 32F=5 | - |

  1. int type( ):回傳一整數，用以表示像素的資料型態。
    表二、type()回傳的整數與Mat資料型態的對應關係
    | - | :C1: | :C2: | :C3: | :C4: |

    | C_64F | 6 | 14 | 22 | 30 |
  2. imread(const char path,int flag):會以flag所對應的讀取方式讀入path路徑的檔案，詳細見下面”從圖檔讀取內容”。
  3. step1():會回傳一列像素總共有幾筆資料，相當於cols*channel。
  4. Iscontinue():回傳像素是否連續
    未完成:這是因為圖像資料每一行中有可能有空，或是0，例如以12位元存資料，但資料內容只有8位元，那就會有4位元為0。可以Iscontinue來檢查資料是否連續。

四、 像素的資料型態:

Mat宣告時，可以透過type來指定陣列的資料型態，命名方式為CV_(bit)(data type)C(channel)，bit代表矩陣中一個數值的位元數，data type代表矩陣中的數值類型，channel代表通道數。

注意!這裡的type是指宣告時的其中一個引數，並不是函數介紹中傳回的數值。Mat物件實際上也沒有type變數可以供使用者調用，在宣告完成之後要改變像素資料型態，必須要透過covert的方式，暫不討論。

  4. 此物件有非常多種宣告方式，這是最基本的格式。

函式原型:
Mat imread( const String& filename, int flags = IMREAD_COLOR );
範例:
Mat imread( (“C:\Users\a2707\Desktop\S__19030095.jpg”);
說明：

  1. 第一個引數是檔案名稱。
  2. 第二個引數指定圖片的格式，可用的選項有三種：
    cv2.IMREAD_COLOR:此為預設值，這種格式會讀取 RGB 三個 channels 的彩色圖片，而忽略透明度的 channel。
    cv2.IMREAD_GRAYSCALE:以灰階的格式來讀取圖片。
    cv2.IMREAD_UNCHANGED:讀取圖片中所有的 channels，包含透明度的 channel。

Visual Studio 2015+OpenCV_安裝方法

相關文章:

  1. 回顧過去

正文開始:

這篇文章記錄了如何下載安裝OpenCV，並在Visual Studio 2015 設定編譯環境，最後試著在Visual studio 2015中運行一段程式碼，測試安裝是否成功。

依序需要完成

  1. 下載OpenCV
  2. 設定電腦環境變數
  3. 設定Viusual studio 2015 編譯環境
  4. 進行測試。

安裝環境

作業系統：Windows 10
開發工具：Visual Studio 2015
OpenCV版本：3.4.XX (安裝位置為C:\opencv)

1.下載安裝OpenCV

下載

到OpneCV官網下載，官網的載點隨著網頁的更新可能會失效，這裡就不直接貼上載點連結，只需要記得找到Releases，然後下載適合你的作業系統的版本，會是一個exe檔。

安裝

執行exe安裝檔，可以安裝在電腦上任意位置，會產生一個opencv資料夾，裡面有build、sources資料夾等等。

2.設定電腦環境變數

  1. 開啟檔案總管-對本機右鍵-內容-(右方)進階系統設定-進階頁籤-(下方)環境變數-使用者變數- path-編輯
  2. 新增以下兩項資料：
    C:\opencv\build\x64\vc14\bin
    C:\opencv\build\x86\vc14\bin

3. 設定Viusual studio 2015 編譯環境

  1. 開啟 Visual Studio 2015 並且新增專案
  2. 於右方方案總管中，對專案點右鍵-屬性
  3. 對「VC++目錄」與「連結器」進行調整
    (1)在x64平台及所有組態下設定
    VC++目錄 →在「Include目錄」增加”C:\opencv\build\include”
    VC++目錄 →在「程式庫目錄」增加”C:\opencv\build\x64\vc14\lib”
    (2)在x64及Release(或Debug)組態下設定
    連結器 →「輸入」 →在「其他相依性」增加“opencv_world34XX.lib”(或“opencv_world34XXd.lib”)

注意!選擇vc14還是vc15是依據你的visual studio版本來決定，甚至更舊visual studio 2013 等也對應不同的vcXX。
目前VC14和VC15與Visual studio的對應關係
Visual Studio 2015 ： vc14
Visual Studio 2017 ： vc15
